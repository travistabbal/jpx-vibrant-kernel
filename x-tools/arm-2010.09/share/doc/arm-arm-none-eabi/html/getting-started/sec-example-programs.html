7.2. Example Programs

Sourcery G++ Lite includes some bundled example programs. You can find the source code for these examples in the share/sourceryg++-arm-none-eabi-examples directory of your Sourcery G++ installation.

The subdirectories contain a number of small, target-independent test programs. You may find these programs useful as self-contained test cases when experimenting with configuring the correct compiler and debugger settings for your target, or when learning how to use the debugger or other features of the Sourcery G++ toolchain.
